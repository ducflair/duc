## Main Components

The ducpy library is organized into several main modules:

- **ducpy.Duc**: Core data structures and formats
- **ducpy.parse**: Functions to parse Duc data and files
- **ducpy.serialize**: Functions to serialize Python objects to Duc format
- **ducpy.classes**: Object-oriented classes for working with Duc data
- **ducpy.utils**: Utility functions and constants

## Getting Started

To get started using the ducpy library, first install it:

```bash
pip install ducpy
```

Then import it in your Python code:

```python
import ducpy as duc

# Access modules directly
duc.parse.parse_duc(...)
duc.serialize.serialize_duc(...)

# Or import specific components
from ducpy.classes import DucElementClass
```

For detailed usage examples, see the [Examples](/docs/libraries/python/examples) page.
