The app state is a simple object that contains the current state of the app.

## Main Components

The main components of the app are the `scope` and the `writingLayer`. These were added on top of the amazing work put behind the `Excalidraw` team and collaborators.
Essentially these give us the capability to have a notion of a these properties on top of the canvas.
